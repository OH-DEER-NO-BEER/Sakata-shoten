# Sakata-shoten

## Overview
坂田商店の電子化を進める．
ユーザはAndroid, iOS, Web上で動作する決済サービスを利用できる．
加えて，管理者は商品登録，棚卸，集金を本システム上で一括して行うことができる．

## 要件定義
- 開発環境
Docker wtih Ubuntu

- 開発言語／ツール
View: Kotlin(iOS, Android Native), React(TypeScript)
Controller: TypeScript (Node.js -> Express)
Model: TypeScript (Node.js -> Express)
